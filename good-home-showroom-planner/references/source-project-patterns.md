# Source Project Patterns

These patterns are distilled from the user's standard template and two project plans. Use them as internal precedent, not as text to copy wholesale.

## Standard Template

Best reusable structure:

政策/品牌背书 -> 系统价值认知 -> 工艺证据验证 -> 服务闭环/成交支持

Standard zones:

0. 门头与导入区
1. 序厅: 品牌与时代背景
2. 好房子总论区
3. 安全系统区
4. 舒适/健康系统区
5. 绿色低碳系统区
6. 智慧系统区
7. 精工工法验证区
8. 服务闭环/成交支持区
9. 结束区/洽谈导流区

Key method:

- Establish four goal weights: brand endorsement, product cognition, sales conversion, communication value.
- Use two tables: management overview and exhibit execution table.
- Standardize exhibit explanation with the 5-sentence formula.
- Provide 15/30/45-minute guide routes and different emphasis for government, industry, first-visit customers, and high-intent customers.
- Configure high/mid/light versions according to budget and flagship level.

## Shenyang CR Land Pattern

Project logic:

- Area: about 123 square meters.
- Structure: compact forced single route with central flexible customization stop.
- Theme: "重归伟大的宁静".
- Core sales problem: city-center living has noise, traffic disturbance, long winter comfort, and uncertainty; the showroom must prove quiet, stable, comfortable long-term living.
- Main route: 入口序启 -> 痛点共鸣 -> 好社区 -> 好产品 -> 好服务 -> 结语.
- Hero proof: acoustic quiet room as the first experiential memory point.
- Conversion mechanism: flexible customization studio with seating, materials, screen, and personal plan discussion.

Reusable principles:

- Do not define a small showroom as a miniature technical museum. Define it as the last space where the customer builds purchase trust.
- "静" can mean acoustic quiet, mental calm, stable systems, service assurance, and family order.
- Use "好社区、好产品、好服务" to simplify complex product systems for sales.
- Avoid technology-blue showmanship. Use warm stone, champagne metal, dark stone, black-gold graphics, and low-saturation green UI.

Memorable line patterns:

- 城市给生活速度，好房子给家庭安静。
- 一门之隔，两个世界。
- 看得见的是美学，看不见的是系统。
- 房子是标准的，生活应该是定制的。

## Changsha Jinmao Pattern

Project logic:

- Area: about 260 square meters.
- Theme: "向江而立，百年恒宅".
- Core sales problem: riverfront luxury is not only scenery; it must answer humidity, river wind, glare, noise, envelope performance, waterproofing, maintenance, and long-term asset stability.
- Narrative: local living proposition -> long-life standard -> craft proof -> daily experience -> long-term stewardship.
- Route: 序厅 -> 长沙江岸居住命题 -> 百年宅总纲 -> 百年宅工法 -> 三懂六境 -> 物业服务 -> 结语.
- Core proof: long-life craft corridor before the lifestyle/technology experience hall.

Reusable principles:

- For luxury projects, use hard logic to support high-end tone, then use soft language to complete emotional recognition.
- Put "long-life standard" before "technology experience" when the premium story depends on time and asset value.
- Treat local climate and site conditions as the proposition, not background.
- Keep property service small but present at the end as long-term trust closure.

Memorable line patterns:

- 一线湘江，不止是风景，也是更高的居住命题。
- 真正的豪宅，不止赢在交付那一天。
- 看不见的系统，决定看得见的长期品质。
- 景观决定第一眼，系统决定每一天。
