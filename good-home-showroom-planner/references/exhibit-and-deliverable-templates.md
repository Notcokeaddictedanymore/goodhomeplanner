# Exhibit And Deliverable Templates

## Proposal Outline

Use for a leader-facing strategy deck or document:

1. 项目理解与核心判断
2. 目标客群与销售任务
3. 策展主题与核心主张
4. 叙事骨架与参观情绪曲线
5. 平面分区与动线策略
6. 重点展项设计
7. 多媒体与互动系统
8. 视觉传达与空间调性
9. 文案体系
10. 销售讲解脚本
11. 材料、工艺与落地控制
12. 实施计划、风险与资料清单

## Single Exhibit Script

Use the 5-sentence sales formula:

1. 大家最担心的是...
2. 很多房子问题出在...
3. 我们在这里用了...
4. 现场您可以看到/听到/测到...
5. 最终带来的好处是...

## Multimedia System Checklist

For each media item, specify:

- Content purpose: atmosphere, explanation, proof, interaction, conversion, or service.
- Hardware: screen size, touch/non-touch, sensor, speaker, lighting, controller, network, power.
- Content form: animation, data dashboard, scenario film, interactive map, UI demo, transparent screen, projection.
- Trigger: auto loop, guide control, visitor touch, sensor, button, door magnet, timed cue.
- Fallback: static graphic, manual explanation, sample display, offline mode.
- Operations: maintenance access, content update path, daily startup/shutdown, safety risk.

## Top-10 Exhibit Priority Logic

Rank exhibits by:

1. Direct sales impact.
2. Pain intensity.
3. Proof visibility.
4. Brand differentiation.
5. Ease of explanation.
6. Feasibility and maintenance.
7. Photo/video spread value.

Do not make all top exhibits screens. Include real samples, tests, models, seating, and a service proof.
